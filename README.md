## 🚀 Ашиглалт

### 1️⃣ Файлыг нээх

Браузерээ нээнэ → `index.html` файлыг сонгоно → дарна

### 2️⃣ Сайт нэмэх

1. **🌐 Сайтууд** табт очно
2. **+ Сайт нэмэх** товчыг дарна
3. Сайтын нэр, URL, тайлбар оруулна
4. **✅ Хадгалах** дарна

### 3️⃣ Сайтын статус шалгах

1. Сайтын картын дээр **🔍 Шалгах** товчыг дарна
2. Систем автоматаар сайтын статус шалгана
3. Үр дүн шигүү гарч ирнэ

### 4️⃣ Мониторинг идэвхжүүлэх

1. **📡 Хянаж байна** табт очно
2. **▶️ Хяналт эхлүүлэх** товчыг дарна
3. Систем автоматаар сайтуудыг мониторинг хийх болно

### 5️⃣ Сайтыг засах эсвэл хасах

- **✏️ Засах** - сайтын мэдээллийг өөрчлөх
- **🗑️ Хасах** - сайтыг системээс устгах
